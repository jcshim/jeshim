# 단계1

### 1. 피코 W 준비 및 포맷 (초기화)

피코 W는 별도의 포맷 프로그램이 필요 없습니다. PC에 연결하면 자동으로 이동식 디스크로 인식됩니다.

1. **BOOTSEL 버튼 누르기**: 피코 W 보드 위에 있는 하얀색 작은 버튼(**BOOTSEL**)을 꾹 누릅니다.
2. **PC 연결**: 버튼을 누른 상태에서 Micro USB 케이블로 PC와 연결합니다.
3. **인식 확인**: PC에서 `RPI-RP2`라는 이름의 새로운 **USB 드라이브**가 나타나면 버튼에서 손을 떼세요. 이것이 피코의 내부 메모리 상태입니다.

---

### 2. 펌웨어(OS 역할) 다운로드

피코 W에서 Wi-Fi를 사용하고 코딩을 하려면 **MicroPython 펌웨어**를 가장 많이 사용합니다.

1. [MicroPython 공식 다운로드 페이지](https://micropython.org/download/RPI_PICO_W/)에 접속합니다.
2. 가장 최신 버전의 **`.uf2`** 확장자 파일을 다운로드합니다.

---

### 3. 복사 및 설치 과정

이 과정이 일반적인 'OS 굽기'에 해당합니다. 아주 간단합니다.

1. 다운로드한 **`.uf2` 파일**을 마우스로 잡습니다.
2. PC에 인식된 **`RPI-RP2` 드라이브로 드래그 앤 드롭(복사)**합니다.
3. 파일 복사가 완료되는 순간, 피코 W는 자동으로 PC에서 연결이 끊기며 재부팅됩니다.
4. 이제 설치가 끝났습니다! 더 이상 `RPI-RP2` 드라이브는 보이지 않으며, 피코는 코드를 실행할 준비가 된 상태가 됩니다.

---

### 4. 코딩 시작하기 (Thonny IDE)

설치된 펌웨어를 이용해 실제로 동작시키려면 PC에 **Thonny**라는 프로그램이 필요합니다.

* **Thonny 설치**: [thonny.org](https://thonny.org/)에서 다운로드 후 실행합니다.
* **설정**: 오른쪽 하단에서 **'MicroPython (Raspberry Pi Pico)'**를 선택합니다.
* **테스트**: 하단 Shell 창에 `print("Hello Pico")`를 입력해서 반응이 오면 성공입니다.

축하합니다! 이제 피코 W는 **MicroPython**이라는 언어를 알아들을 수 있는 상태(뇌가 활성화된 상태)가 되었습니다.

이제 실제로 피코 W에게 명령을 내리고 코딩을 시작할 차례입니다. 가장 쉽고 표준적인 방법인 **Thonny IDE**를 사용하는 단계를 안내해 드릴게요.

# 단계2

### 1. Thonny IDE 설치 (개발 도구)

피코와 대화하기 위해서는 PC에서 'Thonny'라는 프로그램이 필요합니다.

1. [Thonny 공식 홈페이지](https://thonny.org/)에 접속하여 본인의 OS(Windows/Mac)에 맞는 설치 파일을 다운로드하고 설치하세요.
2. 설치가 끝나면 Thonny를 실행합니다.

### 2. 피코 W와 연결 설정

Thonny가 피코 W를 인식하도록 설정해야 합니다.

1. Thonny 오른쪽 하단에 있는 **언어 설정(인터프리터)** 부분을 클릭합니다. (보통 'Python 3.x.x'라고 적혀 있습니다.)
2. 메뉴에서 **MicroPython (Raspberry Pi Pico)**를 선택하세요.
3. 이제 하단 **Shell** 창에 `MicroPython v1.x.x ...` 라는 문구가 뜨면 연결 성공입니다!

---

### 3. 첫 번째 코드 실행 (Hello World)

연결이 잘 되었는지 확인하기 위해 하단 **Shell** 창에 아래 명령어를 입력하고 `Enter`를 눌러보세요.

```python
print("Hello Pico W!")

```

Pico W가 바로 `Hello Pico W!`라고 대답한다면 소통할 준비가 완벽히 된 것입니다.

---

### 4. 내장 LED 깜빡이기 (진짜 코딩 시작)

이제 피코 W 보드에 붙어 있는 작은 LED를 제어해 봅시다. 상단 코드 편집창에 아래 내용을 복사해서 붙여넣으세요. (피코 W는 일반 피코와 달리 LED 핀 설정이 조금 다릅니다.)

```python
from machine import Pin
import time

# 피코 W의 내장 LED는 "LED"라는 이름으로 설정되어 있습니다.
led = Pin("LED", Pin.OUT)

while True:
    led.toggle()      # LED 상태 반전 (켜져있으면 끄고, 꺼져있으면 켜기)
    print("LED 반전!") # 실행 확인용 출력
    time.sleep(1)     # 1초 대기

```

1. 상단의 **[Run] (초록색 재생 버튼)**을 누릅니다.
2. 파일을 어디에 저장할지 묻는 창이 뜨면 **'Raspberry Pi Pico'**를 선택하세요.
3. 파일명은 `main.py`로 입력하고 저장하세요. (파일명이 `main.py`여야 나중에 컴퓨터 연결을 빼고 전원만 연결해도 자동으로 실행됩니다.)
4. 보드 위의 LED가 1초마다 깜빡이는지 확인해 보세요!
